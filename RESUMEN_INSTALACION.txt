┌─────────────────────────────────────────────────────────────┐
│ 🎯 RESPUESTA RÁPIDA                                         │
└─────────────────────────────────────────────────────────────┘

Para instalar ContaFacil en tu celular:

1. Copia este archivo a tu celular:
   📁 ContaFacil.apk

2. Abre el archivo desde tu celular

3. Toca "Instalar"

¡Así de simple! 🎉

┌─────────────────────────────────────────────────────────────┐
│ ⚙️ CONFIGURACIÓN NECESARIA EN TU CELULAR                    │
└─────────────────────────────────────────────────────────────┘

Antes de instalar:

1. Ve a: Ajustes → Seguridad
2. Busca: "Instalar aplicaciones desconocidas"
3. Activa el permiso para tu navegador/gestor de archivos

Nota: En algunos celulares se llama "Fuentes desconocidas"


┌─────────────────────────────────────────────────────────────┐
│ ✅ REQUISITOS                                                │
└─────────────────────────────────────────────────────────────┘

Tu celular debe tener:
✓ Android 7.0 (Nougat) o superior
✓ 20 MB de espacio libre
✓ Permiso de micrófono (para entrada por voz)


┌─────────────────────────────────────────────────────────────┐
│ 🚀 SCRIPTS DISPONIBLES                                      │
└─────────────────────────────────────────────────────────────┘

Si prefieres usar la terminal:

# Instalar en celular conectado por USB
bash install_on_phone.sh

# Regenerar el APK
bash generate_apk.sh

# Ejecutar en emulador
bash run_app.sh
